# Dein "End-to-End Profi-Workflow" für die Lumix S5 II X und Darktable.


PHASE 1: DIE KAMERA "SCHUSSSICHER" MACHEN
Ziel: Ein perfektes digitales Negativ (RAW) erzeugen, das nicht ausbrennt.
1. Das Grund-Setup (Einmalig)
Modus: Wahlrad auf [M] (oder C2).
Format: [RAW] (Niemals nur JPG).
Sicherheits-Anzeige: [Konst. Vorschau] -> [ON] (Damit du keine schwarzen Bilder schießt).
Überbelichtungs-Warner: [Zebra 1] -> [100% oder 95%] (Statt Waveform).
ISO-Logik: [Dual Native ISO] -> [AUTO] | [ISO-Obergrenze] -> [6400/12800].
2. Die Aufnahme (Am Aquarium)
Messmethode: [Lichterbetont] (Das Glint-Icon). Schützt die hellen Schuppen.
Belichtung:
Blende: f/4.0 - f/5.6 (Gibt dem Fisch Tiefe).
Verschlusszeit: 1/500 oder schneller (Friert Flossen ein).
ISO: Macht die Kamera automatisch.
Kreativer Spielraum:
Bewegung: Gehe auf 1/125s für leichte Bewegungsunschärfe (dynamisch) oder 1/1000s für kompletten "Freeze".
Tiefe: f/2.8 für extreme Isolation (nur das Auge scharf), f/8 für ganzes Aquarium scharf.
PHASE 2: DATEN-HYGIENE (Der unsichtbare Teil)
Ziel: Kein Datenverlust.
3. Import & Verwaltung
Importiere die Bilder in Darktable.
Goldene Regel: Verschiebe Bilder niemals im Windows Explorer. Nutze immer Darktable Leuchttisch -> [ausgewählte Bilder] -> [verschieben].
Ignoriere die .xmp Dateien, Darktable pflegt sie automatisch.
PHASE 3: ENTWICKLUNG (Darktable Scene-Referred)
Ziel: Vom dunklen RAW zum Kunstwerk. Halte diese Reihenfolge ein.
4. Modul: Belichtung (Das Licht einschalten)
Aktion: Ziehe den Regler massiv ins Plus (+2 bis +4 EV), bis die Helligkeit stimmt.
Kreativer Spielraum: Lasse es für Aquarien leicht unterbelichtet (-0.5 EV) für eine mystische "Tiefsee"-Stimmung.
5. Modul: Objektivkorrektur & Entrauschen (Die Pflicht)
Aktion: Beides einfach anschalten.
Kreativer Spielraum: Keiner. Das ist technische Hygiene.
6. Modul: Zuschneiden (Die Komposition)
Aktion: Wähle Format [16:9] (Kino-Look). Ignoriere Pixelzahlen!
Kreativer Spielraum: Platziere das Auge des Fisches auf einer Schnittlinie (Drittel-Regel), nicht stumpf in der Mitte.
7. Modul: Sigmoid (Die Entwicklung)
Aktion: Ersetzt "Filmic RGB". Regle den Kontrast (ca. 1.5 bis 1.8) und Schiefe (Helligkeitsverteilung).
Kreativer Spielraum: Hoher Kontrast = Action/Drama. Niedriger Kontrast = Sanft/Traumhaft.
8. Modul: Farbbalance RGB (Der Look)
Reiter "Allgemein": Ziehe [Globale Lebendigkeit] hoch (+20%).
Reiter "4 HSL" (Profi-Grading):
Schatten -> Farbton Blau, Chroma hoch (5-10%).
Lichter -> Farbton Orange, Chroma hoch (5-10%).
Kreativer Spielraum: Das ist deine Handschrift. "Teal & Orange" ist modern. "Grüne Schatten" wirken wie "Matrix".
9. Modul: Effektgruppe (Das Finish)
Vignettierung: Weicher Radius, leicht abdunkeln. Lenkt den Blick.
Körnung: ISO 400 Preset. Macht das Bild "griffig" und kaschiert Glätte.
Kreativer Spielraum: Starke Vignette & Korn = "Retro/Analog". Keine Effekte = "Clean/Digital".
PHASE 4: EXPORT (Das Finale)
Ziel: Maximale Schärfe für den Desktop.
10. Export-Einstellungen
Einheit: WICHTIG -> Auf [Pixel] stellen (nicht Zoll!).
Größe: 1920 x 1080 (oder deine Monitorauflösung).
Resampling: [hochqualitatives Resampling] -> [ja].
Format: JPEG (8-bit), Qualität 95.
Dateiname: Hier darfst du umbenennen (z.B. Fisch_Final_Wallpaper).
Du bist jetzt sicher.
Du hast den vollen RAW-Dynamikumfang genutzt, die Daten nicht zerstört (non-destruktiv) und das Bild mathematisch perfekt auf deinen Monitor geschneidert.
